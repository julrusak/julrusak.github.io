Looking at a Ruby enumerable

Map it!

July 11, 2015

Using enumerables in Ruby is not only incredibly efficient, it's also super easy! I mean, Ruby was designed to be easier to read, so that's not a surprise.

The map method can be quite useful when you want to modify the objects in an array. Using map, a new array will be returned with the results of running the code in the block once for every element. 

      array = [1,2,3,4]
      new_array = array.map { |i| i + 1}

      puts array.join(", ")  #=> 1, 2, 3, 4
      puts new_array.join("' ") #=> 2, 3, 4, 5
The map method will create a new array and complete the modification made in the block on each element of the array.

To further break down what is going on; 

      new_array = array.map { |each element| code to run}
I hope that clears it up for you!

Happy Coding!
